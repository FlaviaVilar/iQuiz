# iQuiz

An iOS quiz application built using UIKit and Storyboard.

This app was developed as a hands-on project during the course **"Praticando a orientação a objetos em Swift"**, covering the basics of iOS app development.

## Features

✅ Built with **UIKit**  
✅ Layouts created with **Storyboard**  
✅ Navigation between multiple screens  
✅ Responsive UI using **Auto Layout** and constraints  
✅ Data modeling with **Swift structs**

## Requirements

- iOS 15.6+  
- Xcode 13+  
- Swift 5

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/iQuiz.git
    ```

2. Open the project in Xcode:

   ```bash
open iQuiz.xcodeproj
    ```

3. Build and run the project on the simulator or a physical device.
